# GitHub Copilot de Graça para Estudantes

Se você está estudando programação ou quer acelerar seu aprendizado, existe um benefício pouco conhecido que pode fazer muita diferença: o acesso gratuito ao GitHub Copilot através do GitHub Education.

## O que é o GitHub Education?

O GitHub Education é um programa criado pelo próprio GitHub para ajudar estudantes a aprenderem com ferramentas profissionais.

👉 https://education.github.com/pack

## O que é o GitHub Copilot?

O GitHub Copilot é uma ferramenta de inteligência artificial que funciona como um assistente de programação.

## Vale a pena?

Sim — principalmente se você está começando ou quer evoluir mais rápido na programação.
